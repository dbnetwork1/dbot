Moderation Commands

Moderators may run these commands incase Watchdog misses someone!

w!kick

Kicks a user from the server

w!ban

Bans a user from the server

w!clear

Delete a given amount of messages

w!mute

Mutes a user from all channels for a given time 

w!unmute

Unmutes a user

w!warn

Adds an infraction to a user

w!infractions

View a users infractions

w!infractions remove

Remove a infraction from a user

w!snipe

View the last deleted message in the channel

w!say

Speak as the bot
